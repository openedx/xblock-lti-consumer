LTI 1.3 Launches can only be performed from the LMS.

The current implementation of the LTI 1.3 XBlock does not support LTI Advantage services . Therefore, there is no mechanism for LTI Tools to pass back grades.

To set up the LTI integration, you need to register the LMS in the tool with the information provided below.

Client: ${client_id}

Deployment ID: ${deployment_id}

Keyset URL: ${keyset_url}

OAuth Token URL: ${token_url}

OIDC Callback URL: ${oidc_callback}

Test LTI Launch: - The launch only works if the block is published.
